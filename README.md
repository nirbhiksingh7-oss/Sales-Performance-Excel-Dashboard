# Sales Performance Excel Dashboard

## Project Goal
Analyze sales performance by manager and product over time and build an interactive Excel dashboard for decision-making.

## Tools Used
- Microsoft Excel
- Tables & Formulas (VLOOKUP, IF, COUNTIFS)
- Pivot Tables / Power Pivot
- Charts & Dashboard Design

## Project Structure
- Raw sales data with calculated metrics
- Manager and product lookup tables
- Performance analysis by month and manager
- Summary KPIs using Power Pivot
- Interactive dashboard

## Key Insights
- Identified top-performing managers by total sales and net proceeds
- Highlighted best-selling products
- Tracked monthly sales trends
- Analyzed commission and profitability
